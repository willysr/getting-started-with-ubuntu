    Assessment for Module 2 - ROS configuration and structure

    ROS configuration and structure

    This assignment is to help you self-assess your comprehension on ROS. Read carefully each question. Once answered, you will receive feedback. Good luck!

    Q1

    The command source /opt/ros/melodic/setup.bash is for

    1. will set the environment variables so that all ROS commands are available at command line.
    2. will install ROS.
    3. will add the ROS repository for the apt command.
    4. the file does not exist.

    Q2

    A ROS package is

    1. a directory containing files, including configuration files, executables, source code, etc, typically organized to provide specific robot functionalities.
    2. a zip file.
    3. a part of the robot hardware.
    4. provides only the functionality of Simultaneous Localization and Mapping.

    Q3

    The ROS packages are searched within

    1. the directories listed in ROS_PACKAGE_PATH
    2. in /opt/ros/melodic/
    3. ~/ros_workspace/src/
    4. /opt/ros/melodic/share

    Q4

    To install the package ‘gmapping’ in ROS melodic through apt, the full command to be typed on the terminal is

    Possible answers:
    • sudo apt install ros-melodic-gmapping

    Q5

    Some of the ROS benefits include (select all that apply)

    * Software Reuse
    * Rapid testing
    * Distributed computation
    * it is a simple programming language.
    * it is an integrated development environment (IDE)
